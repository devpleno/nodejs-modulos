# NodeJS Módulos

Exemplos de módulos em NodeJS.

## Para executar:

Após ter o NodeJS instalado (http://www.nodejs.org), basta no diretório da aplicação digitar:

Exemplo 01:

- node app.js

Exemplo 02:

- node app2.js
